# Student_Performance_Analysis

Tools Used:
Excel,
Power BI

**Project Overview:**
This project analyzes student performance data to understand the impact of factors like study time, parental education, and lifestyle on academic results.

**Key Visualizations:**
Student score distribution,
Study time vs performance,
Gender-based performance comparison,
Parental education impact,
Attendance vs scores.

**Key Insights:**
 Students who study more hours tend to score higher, 
 Parental education has a positive impact on performance, 
 Female students slightly outperform male students, 
 Attendance is directly linked to higher scores, 
 Some students underperform despite high study time → possible external factors.
